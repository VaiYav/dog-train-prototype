# Feature: Швидкий лог дня (`005-quick-log`)

> lite forge · демо в основному прототипі (вкладка «Догляд»).

## Дуже коротко
**Що:** пишеш/диктуєш «погуляли 30 хв, дав ліки» — система розпізнає й заносить події догляду. **Де:** поле **«✍️ Швидкий лог»** у вкладці «Догляд». **Як:** `parseLog(text)` (prod — LLM, proto — ключові слова) → **підтвердження** → запис у `Completion`/`CareEvent`. **Навіщо:** менше тертя в логуванні → більше залучення.
