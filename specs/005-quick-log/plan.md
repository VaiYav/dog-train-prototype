## Логіка
```
text → parse (prod: LLM; proto: keywords) → events[] (Completion/CareEvent)
показати для ПІДТВЕРДЖЕННЯ → on confirm: write; track('ai_log_parsed',{n})
```

## Метрики / guardrails
**Primary:** залученість логування / D7. **Guardrails:** точність парсингу (виправлення), приватність голосу, без авто-запису.

## Prod vs прототип
**Prod:** LLM-парсер + підтвердження. **Прототип:** `parseLog(text)` за ключовими словами ([прототип](../../prototype/index.html), вкладка «Догляд»).
