# Feature: AI челенджі/місії (`004-ai-missions`)

## Дуже коротко
**Що:** щотижнева персональна місія («навчіть нову команду»). **Де:** картка **«🎯 Місія тижня»** на вкладці «Прогрес». **Як:** `getPersonalizationContext()` (ціль + опановане + вік/розмір) → у проді LLM, у прототипі — генератор; rules-безпека. **Навіщо:** новизна + ціль → звичка.
