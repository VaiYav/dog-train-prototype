# Competitor Analysis: DogTrain (базовий продукт)

> Згенеровано: 2026-06-09 · Дименсія: 5 застосунків · Фокус — як конкуренти будують онбординг, персональний план і щоденний цикл догляду.

## Executive Summary

Ринок dog-апів поділений на два табори: **тренувальні** (Dogo, Puppr, Pupford, Woofz — сильні в персональних програмах навичок і відео-уроках) і **догляд/здоров'я** (11pets — сильний у нагадуваннях про щеплення, грумінг, вагу, документи для ветеринара). **Майже ніхто не закриває єдиний щоденний цикл «здоров'я + активність + догляд + тренування» в одному легкому потоці** — це і є позиційний розрив DogTrain.

Найсильніший онбординг — у **Dogo**: коротке опитування (порода, вік, поведінка) → миттєвий персональний план → щоденні завдання з нагадуваннями. Це фактично референс для нашого фундаменту. Водночас тренувальні апи слабкі в обліку здоров'я, а 11pets — навпаки: потужний облік, але **немає щоденного «що зробити сьогодні»** і мотиваційної петлі. Гіпотеза H1 із Problem Discovery (розрив у єдиному циклі) **підтверджується**.

## Competitors Analyzed

### 1. Dogo — [5/5] (референс онбордингу й щоденного циклу)
- **Фіча:** персональні тренувальні програми за результатами онбординг-опитування (вік, порода, поведінка); 100+ відео-уроків, 12 програм, вбудований клікер, нагадування про здоров'я, фідбек тренера, сертифікати досягнень.
- **Core UX pattern:** опитування → згенерований план → щоденні кроки з відмітками й прогресом.
- **Differentiator:** поєднання персоналізації + щоденної звички + фідбеку тренера.
- **Access:** Freemium; $9.99/тиж, $29.99/міс, $49.99/3міс; 7-денний trial, −50% на перший період.
- **Sentiment:** + персоналізація і структура; − агресивний paywall одразу після онбордингу.

### 2. Puppr — [4/5]
- **Фіча:** 100+ уроків (Sara Carson), live-чат із тренерами, **система нагадувань про тренування і туалет**, трекінг прогресу.
- **Core UX pattern:** бібліотека уроків + нагадування + ручні відмітки.
- **Differentiator:** живі тренери в чаті.
- **Access:** $12.99/міс, $99.99/рік, 7-денний trial.
- **Sentiment:** + якість контенту; − цінність прив'язана до підписки.

### 3. Pupford — [4/5]
- **Фіча:** безкоштовний 30-денний курс (Zak George), щоденні поради, **трекінг поведінки з відмітками «день виконано»** і оцінкою прогресу.
- **Core UX pattern:** структурований курс із днями → відмітка дня → трекінг.
- **Differentiator:** сильний безкоштовний вхід (low friction), денний горизонт «один день курсу».
- **Access:** значна частина безкоштовно + платні розширення.
- **Sentiment:** + безкоштовність і простота; − менше персоналізації під конкретну собаку.

### 4. 11pets — [3/5] (референс обліку здоров'я/догляду)
- **Фіча:** облік здоров'я — нагадування про ліки, щеплення, дегельмінтизацію, ветвізити; графік гігієни (купання, кігті, зуби, шерсть); вага/віктальні; фото-документація симптомів; мульти-пет; шеринг із ветеринаром.
- **Core UX pattern:** картки записів + календар нагадувань (більше «журнал», ніж «щоденна звичка»).
- **Differentiator:** глибина медичного обліку й шеринг із ветом.
- **Access:** Free або $4.99/міс premium.
- **Sentiment:** + повнота обліку; − **немає щоденного «що зробити сьогодні»** і мотивації; інтерфейс «адмінський».

### 5. Woofz / Zigzag — [3/5]
- **Фіча:** персональні тренувальні плани для цуценят, щоденні завдання, нагадування (Zigzag — із підтримкою кінологів).
- **Core UX pattern:** план за віком/етапом → щоденні завдання.
- **Differentiator:** вузький фокус на цуценятах і ранній соціалізації.
- **Access:** Freemium / підписка.
- **Sentiment:** + структурований шлях для новачка; − звужено до цуценят.

## Common Patterns (table stakes — must-have для фундаменту)

- Онбординг-опитування про собаку (порода, вік, інколи поведінка) → **персональний план** на виході.
- **Денний/покроковий горизонт** із відмітками виконання (а не лише великий план).
- Трекінг прогресу (відсоток курсу / стрік / бейджі).
- Нагадування (тренування, туалет, здоров'я).
- 7-денний trial + підписка одразу після онбордингу.

## Differentiation Opportunities (де ніхто не сильний — наша перевага)

1. **Єдиний щоденний цикл** «здоров'я + активність + догляд + тренування» в одному екрані «Сьогодні» (тренувальні апи ≠ облік здоров'я; 11pets ≠ щоденна звичка). **Найвищий impact.**
2. **Антипаттерн раннього paywall.** Конкуренти продають одразу після онбордингу — можливість дати «першу перемогу» безкоштовно й утримати, перш ніж монетизувати.
3. **Легкість щоденного ведення.** Усі сильні в контенті, але важкі у щоденному UX — простий, «на один дотик» денний чек-ін.
4. **Гуманна мотивація** (стрік/прогрес без покарання за пропуск) замість агресивних нагадувань.

## Top 3 Reference Implementations

1. **Dogo** — еталон зв'язки «онбординг-опитування → персональний план → щоденні відмітки + нагадування». Беремо як референс для фундаменту, але **прибираємо ранній paywall**.
2. **Pupford** — еталон low-friction входу й денного горизонту «один день за раз» (підказка для retention-фічі Daily Focus).
3. **11pets** — еталон моделі даних догляду/здоров'я (нагадування, гігієна, вага) для блоку «догляд» нашого плану.
