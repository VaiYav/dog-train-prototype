# User Interview Script: DogTrain — базовий догляд

> **Мета:** перевірити, що проблема «немає одного місця для персонального щоденного догляду й видимого прогресу» реальна й достатньо часта, щоб її розв'язувати.
> **Тривалість:** 30 хв · **Формат:** 1:1, напівструктуроване · **Кого кличемо:** власники собак, які завели собаку за останні 12 міс., користуються смартфоном щодня.

## Розігрів (5 хв)
1. Розкажіть, як виглядає ваш звичайний день із собакою — від ранку до вечора.
2. Як давно у вас собака? Порода, вік?
3. Чим зараз користуєтесь, щоб не забути про догляд/активність/візити (нотатки, календар, апки, голова)?

## Дослідження проблеми (10 хв)
4. Згадайте останній раз, коли ви не були впевнені, чи правильно дбаєте про собаку. Що це було?
5. Що в догляді дається найважче — фізично чи морально?
6. Що ви робите, коли не знаєте відповіді (скільки гуляти, як привчати, коли щеплення)?
7. Скільки часу за тиждень іде на «зрозуміти, що треба робити» (гуглення, питання, пошук)?
8. Чи був момент, коли ви закинули якусь корисну рутину? Чому?

## Зондування поточних рішень (10 хв)
9. Пробували dog-апи (Dogo, Puppr тощо)? Що сподобалось / не зайшло?
10. Чого найбільше бракує в тому, чим користуєтесь сьогодні?
11. Якби була «чарівна паличка» і можна змінити ОДНУ річ у догляді — що б це було?

## Валідація рішення (5 хв)
12. Якби застосунок щодня показував 1–3 конкретні завдання саме для вашої собаки і показував прогрес — як би це змінило ваш день?
13. За шкалою 1–10, наскільки це було б важливо?
14. Що б змусило вас закинути такий застосунок уже на 2–3 день?  *(прямий зонд на корінь відтоку)*

## Завершення
15. Хто ще у вашій родині/колі займається собакою?
16. Чи є щось важливе, про що я не запитав?

## Рубрика оцінювання (заповнити після кожного інтерв'ю)

|---|---|---|
| Частота проблеми | | |
| Рівень болю сьогодні | | |
| Якість обхідних шляхів (нижче = краще для нас) | | |
| Інтерес до рішення | | |
| Ризик кинути на 2–3 день (питання 14) | | |
| **Разом** | /25 | |

> ≥18/25 → сильний сигнал, рухаємось у Research.
> 12–17 → помірний, потрібно ще 3–5 інтерв'ю.
> <12 → слабкий, переглянути рамку проблеми.

**Окремий фокус цього скрипту:** питання 8 і 14 цілять прямо в проблему утримання (відтік 2–3 день) — їхні відповіді живлять гіпотези Фази 2 (аналіз відтоку).
