## Feature Description
Базовий продукт DogTrain (профіль собаки → 8-кроковий онбординг → rules-based план → «Сьогодні» з 3 завданнями → відмітки/прогрес → нагадування догляду/здоров'я → trial одразу), реалізований як клікабельні HTML-прототипи (mobile-first). Фундамент, на який у Фазі 3 стане retention-MVP проти відтоку.

## Decisions baked in
