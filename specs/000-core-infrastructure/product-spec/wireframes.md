## Екран 1 — Онбординг (крок X/8)

**Purpose:** зібрати персоналізацію за 8 односкрокових екранів, не вбивши TTV.

```
┌───────────────────────────────┐
│  ←            ▓▓▓▓▓░░░  4/8     │  ← прогрес-бар (видно, що близько)
├───────────────────────────────┤
│                               │
│   Який рівень активності       │
│   у вашого песика?             │
│                               │
│   ┌───────────────────────┐   │
│   │ 🛋  Спокійний          │   │  ← великі тач-таргети ≥44px,
│   ├───────────────────────┤   │     одне рішення на екран
│   │ 🚶  Помірний       ✓   │   │
│   ├───────────────────────┤   │
│   │ ⚡  Дуже активний      │   │
│   └───────────────────────┘   │
│                               │
│        [  Далі  ]              │  ← кнопка в зоні великого пальця
│        Пропустити             │     (для необов'язкових кроків)
└───────────────────────────────┘
```
**Components:** прогрес-бар, питання, варіанти-картки, Далі/Пропустити. **States:** default · обрано · skip-able.

## Екран 3 — «Сьогодні» (3 завдання) ★ головний

**Purpose:** дефолтний екран; чіткий денний фокус (не стіна тижня).

```
┌───────────────────────────────┐
│  Привіт! Рекс 🐶     🔥0  ⚙    │  ← (стрік = 0; стрік "оживе" у Фазі 3)
├───────────────────────────────┤
│   Сьогодні · Пн                │
│      ◌ 0/3 виконано            │  ← кільце прогресу
│                               │
│  ┌─────────────────────────┐  │
│  │ ☐ 🚶 Коротка прогулянка   │  │  ← перше = тривіальне
│  │    15 хв · чому це важливо │  │     (гарантує першу перемогу)
│  ├─────────────────────────┤  │
│  │ ☐ 🦴 Тренування «сидіти»  │  │
│  │    5 хв                   │  │
│  ├─────────────────────────┤  │
│  │ ☐ 💧 Свіжа вода + огляд лап│  │
│  └─────────────────────────┘  │
│                               │
│   Переглянути весь тиждень →   │  ← вторинна дія (не дефолт)
├───────────────────────────────┤
│  [Сьогодні] [Догляд] [Прогрес] │  ← таб-бар
└───────────────────────────────┘
```
**States:** default · 0/3 · частково · all-done · empty · offline.

## Екран 7 — Trial / soft paywall

**Purpose:** trial активний одразу; paywall **не блокує** першу перемогу.

```
── під час trial (день 1–7): ──        ── після trial: ──
┌───────────────────────────────┐   ┌───────────────────────────────┐
│  Сьогодні ... (повний доступ)  │   │   Trial завершено              │
│  ┌──────────────────────────┐ │   │   Продовжуй піклуватись про    │
│  │ ✨ Trial: 6 днів лишилось │ │   │   Рекса з Premium              │
│  └──────────────────────────┘ │   │   [ Оформити ]  [ Пізніше ]    │
└───────────────────────────────┘   │   (прогрес/історія лишаються)  │
                                     └───────────────────────────────┘
```
> Час показу жорсткого paywall (одразу vs кінець trial) — **A/B-кандидат**, дефолт — після першої перемоги.
