# Acceptance Checklist: Daily Focus loop (`001-daily-focus`)

**Purpose**: умови, за якими вважаємо фічу реалізованою правильно.
**Created**: 2026-06-09 | **Feature**: [spec.md](./spec.md)

## A. Нагадування
- [ ] CHK001 Контекстний opt-in зʼявляється **після першої перемоги**, не на старті
- [ ] CHK002 Доступний вибір часу (Ранок/День/Вечір/свій); дефолт визначено
- [ ] CHK003 Шле **рівно 1** нагадування/день у обраний час
- [ ] CHK004 Текст нагадування відображає завдання дня; тап → «Сьогодні» (deep-link)
- [ ] CHK005 Опт-аут нагадувань — у 1 тап; після `day_completed` нагадування не шлеться

## B. Стрік (гуманний)
- [ ] CHK006 Стрік++ при **першому** виконаному завданні за день; не змінюється при наступних
- [ ] CHK007 Мікросвято на **3** і **7** днів
- [ ] CHK008 Пропуск дня: freeze-токен рятує серію (якщо є), інакше **мʼякий** рестарт без різкого «−усе»
- [ ] CHK009 Freeze-токен поповнюється (1/тиждень)
- [ ] CHK010 Control **не бачить** активного стріку (🔥0) і engagement-нагадування

## C. Експеримент / аналітика
- [ ] CHK011 `getVariant('daily_focus')` ділить нових 50/50; спліт перевірено (no SRM)
- [ ] CHK012 Усі нові події (`reminder_*`, `streak_*`) емітяться з тегом `variant`
- [ ] CHK013 Дашборд показує D0→D2 воронку, retention-матрицю та всі guardrails
- [ ] CHK014 Pre-registration A/B існує (гіпотеза, Primary, guardrails, MDE, sample size)

## D. Гуманність / A11y (constitution III, V)
- [ ] CHK015 Жодного покарання за пропуск; легко вимкнути нагадування
- [ ] CHK016 Стрік/мікросвято — не лише колір (іконка + число + текст)
- [ ] CHK017 `prefers-reduced-motion` вимикає конфеті/анімації стріку

## E. Guardrails (не гірше control)
- [ ] CHK018 Opt-out/uninstall не зростають; activation/onboarding completion не падають
- [ ] CHK019 Сер. завдань на активного не падає; crash-free/Today-load стабільні

## Notes
