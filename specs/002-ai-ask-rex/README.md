# Feature: «Запитай про Рекса» (`002-ai-ask-rex`)

> AI-чат Q&A · lite forge · демо в основному прототипі.

## Дуже коротко
**Що:** чат, де питаєш про свою собаку й отримуєш відповідь під неї. **Де:** плаваюча кнопка **«💬 Запитай про Рекса»** в прототипі → оверлей-чат із підказаними питаннями. **Як:** `getPersonalizationContext()` + куратована база → у проді LLM, у прототипі — grounded відповіді за темами. **Безпека:** медичні питання → **дисклеймер «до ветеринара», без діагнозу**. **Навіщо:** утиліта + довіра → привід відкривати.
