# Feature: AI «порада дня» (`003-ai-daily-tip`)

> Перша AI-фіча (lite forge) · RICE-lite 731 · реалізовано в основному прототипі.

## Дуже коротко
**Що:** персональна порада про собаку щодня. **Де:** картка «💡 Порада дня» **угорі екрана «Сьогодні»**. **Як:** береться профіль (порода-група/вік/ціль) + прогрес (стрік/виконане/пропущене) → у проді LLM, у прототипі — імітація правилами → 1–2 речення, ↻ оновити, 👍/👎. **Навіщо:** свіжий контент-привід повертатись (ретеншн).
